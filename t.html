Developer API Access

Integrate CBTHost.com with your existing systems using our comprehensive REST API. Verify student results, fetch performance data, and manage institutional accounts programmatically.

  • Secure API key authentication
  • Real-time student result verification
  • Comprehensive documentation and SDKs
  • Webhook support for event notifications
JavaScript Example 
// Verify student results via API
const verifyStudentResult = async (studentId, quizId) => {
  const response = await fetch('https://api.cbthost.com/api/verify', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({ studentId, quizId })
  });
  return await response.json();
};
